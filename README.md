# A database system for a large franchised real estate company

## ❓How to run SQL files 
```
$ pip3 install sqlite3 
$ sqlite3 
$ sqlite > .read file 
```

## ❓How to run Unit tests 
```
$ pip3 install virtualenv 
$ virtualenv env
$ source env/bin/activate 
$ python unit_tests.py 
```
